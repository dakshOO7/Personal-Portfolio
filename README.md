# Personal Portfolio

My personal portfolio built with **React** and **Vite** to showcase my education, skills, projects, and experience.

## Features
- Dark-themed modern UI
- Tab navigation (Education, Experience, Skills, Projects, Resume)
- Responsive for desktop & mobile
- GitHub project links
- Resume preview inside the site

## Tech Stack
- React
- Vite
- Tailwind CSS
- JavaScript

## Run Locally
```bash
npm install
npm run dev
